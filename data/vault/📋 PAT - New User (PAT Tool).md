# 📋 PAT — New User (PAT Tool)

Procedure to create a new user in the PAT Tool, assign security groups, and add the user to a team box.

---

## Procedure

### STEP 1 — Search Existing User

On the PAT Tool, navigate to **Admin → Membership → Users**. Click on **Search** and enter the username you are going to create.

![[assets/doc-pat-new-user-search.png]]

> ⚠️ **NOTE:** Always search first to verify the user does not already exist.

### STEP 2 — Create New User

If the user was not found, click on **"New User"**.

![[assets/doc-pat-new-user-create.png]]

### STEP 3 — Fill User Information

Fill in the required information for the new user:

![[assets/doc-pat-new-user-form.png]]

> ⚠️ **IMPORTANT:** Uncheck **"User Must Change Password at Next Login"**.

### STEP 4 — Add Security Groups

Add the following Security Groups to the new user:

![[assets/doc-pat-new-user-groups.png]]

### STEP 5 — Assign to Team Box

Once you have finished adding the Security Groups, navigate to **Quotes → Teams**. Search for the Team Box where you will add the user.

![[assets/doc-pat-new-user-team.png]]

> 💡 **Example:** In this case we add the user to the Team Box **CC65**.

---

## Quick Checklist

- [ ] Search user first (verify it doesn't exist)
- [ ] Create new user with correct credentials
- [ ] Uncheck "Must Change Password at Next Login"
- [ ] Add all required Security Groups
- [ ] Assign user to correct Team Box
